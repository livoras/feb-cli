feb-cli
======================================

Command Line of [FEB](https://github.com/livoras/feb).

Using [FEB](https://github.com/livoras/feb) for front-end project boilerplate. This project is to provide a command-line tool to fast create and initialize a project.

### Installation

    npm install feb-cli -g

### Create a project.

    feb new hello-world

Then run:

    cd hello-world && npm install

Project structure is organized exactly the same as [FEB](https://github.com/livoras/feb)(feb-cli just copy all the files of FEB actually). You can read the README of FEB for more details.
